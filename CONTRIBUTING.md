# Contributing to Graphify

Thank you for your interest in making Graphify even better! We welcome all contributions—big or small. Here’s how you can get involved:

---

## 📢 Code of Conduct

We’re committed to a welcoming, inclusive community. Please read our [Code of Conduct](CODE_OF_CONDUCT.md) before contributing.

---

## 🐞 Reporting Bugs

Found a bug? Please [open an issue](https://github.com/iiXXiXii/Graphify/issues) and include:
- A clear, descriptive title
- Steps to reproduce
- What you expected to happen
- What actually happened
- Screenshots (if helpful)
- Your environment (OS, browser, etc.)

---

## 💡 Suggesting Features

Have an idea? We’d love to hear it! [Open a feature request](https://github.com/iiXXiXii/Graphify/issues) and tell us:
- What the feature is
- Why it’s useful
- Any ideas for implementation

---

## 🚀 Submitting Pull Requests

1. **Fork** this repo
2. **Create a branch**: `git checkout -b feature/your-feature`
3. **Make your changes**
4. **Test** to make sure everything works
5. **Commit**: `git commit -m 'Add your feature'`
6. **Push**: `git push origin feature/your-feature`
7. **Open a Pull Request**

---

## 🛠️ Development Setup

See the [README.md](README.md) for setup instructions.

**Project Structure:**
- `frontend/` – Angular app
- `backend/` – NestJS app
- `graphify-cli/` – CLI tool
- `shared/` – Shared utilities
- `prisma/` – Database schema

---

## 🧑‍💻 Coding Standards
- Use TypeScript best practices
- Write clear, descriptive commit messages
- Keep code modular and well-commented
- Add tests for new features

---

## 🧪 Testing
- Run `bun test` to check all tests
- Make sure all tests pass before submitting
- Add tests for any new code

---

## 📦 Release Process

We use [Semantic Versioning](https://semver.org/):
- **MAJOR**: Breaking changes
- **MINOR**: New features
- **PATCH**: Bug fixes

---

## 📚 Documentation
- Update docs for any API or UI changes
- Use JSDoc for functions/classes
- Keep the README and other docs up-to-date

---

## ❓ Questions?

Open an issue or reach out to the maintainers. Thanks for helping make Graphify awesome!
