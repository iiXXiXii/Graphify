## 🚀 What is Graphify?

Graphify lets you make your GitHub profile truly yours! Design custom contribution patterns—spell your name, create geometric art, or highlight achievements. With Graphify, you’re in control of your contribution graph.

---

## ✨ Features

- **Visual Pattern Designer:** Intuitive grid editor for stunning patterns
- **Live Preview:** See your design before you commit
- **Schedule Manager:** Plan when your contributions appear
- **GitHub Integration:** Securely connect and automate your commits

## 🛠️ Quick Start

Want to run Graphify locally? It’s easy:

```bash
# Clone the repository
git clone https://github.com/iiXXiXii/Graphify.git
cd Graphify

# Install dependencies with Bun
bun install

# Start the app
bun run dev:frontend
```

Visit [http://localhost:4200](http://localhost:4200) in your browser.
